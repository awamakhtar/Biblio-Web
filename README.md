# Bibliothèque en Ligne - Projet Final
Ce projet consiste à créer un site web interactif pour une bibliothèque en ligne, permettant de rechercher, ajouter, modifier et supprimer des livres, ainsi que de gérer une liste de lecture.
Il met en pratique les compétences en HTML, CSS, JavaScript, PHP et MySQL dans le cadre de la formation délivré par l'OIF

# Fonctionnalités principales
 # Recherche de livres
Formulaire de recherche par titre ou auteur
Affichage des résultats sur une page dédiée
Bouton pour consulter les détails du livre

 # Détails d'un livre
 Affichage complet : titre, auteur, description, maison d’édition, nombre d’exemplaires
 Affichage complet : titre, auteur, description, maison d’édition, nombre d’exemplaires

 # Liste de lecture
 Affichage des livres ajoutés par l’utilisateur
 Suppression de livres de la liste
